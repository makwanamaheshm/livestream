REQUIREMENTS:
=============
1. Python 3.8 or higher
2. FFmpeg (must be installed and in PATH)
3. OpenAI API Key (for Intelligent Auto-Sync)

INSTALLATION:
=============
1. Install Python dependencies:
   pip install -r requirements.txt

2. Install FFmpeg:
   - Windows: Download from https://ffmpeg.org/download.html
   - Mac: brew install ffmpeg
   - Linux: sudo apt install ffmpeg

3. Get OpenAI API Key:
   - Go to https://platform.openai.com/api-keys
   - Create a new API key
   - Add it in Settings within the app

USAGE:
======
1. Run the application:
   - Windows: Double-click start.bat
   - Mac/Linux: ./start.sh or python3 ai_studio_pro.py

2. Add your audio/voice file

3. Add your images (folder or individual)

4. Click "START AUTO-SYNC" - AI will:
   - Transcribe your audio
   - Analyze your images
   - Match images to voice automatically
   - Generate captions with word timings

5. Preview and export your video!

KEYBOARD SHORTCUTS:
